Supplemental Data

Scripts:
AssessPseudogenes_v#.R = Main R script to plot most figures. Set the working directory to run in R.
ConvertToLineDiagram_v#.R = R script to plot the output of breseq.

Data:
all-prokaryotes_v#.RData = Dataset downloaded from https://www.ncbi.nlm.nih.gov/genome/browse/
anno_v#.RData = Annotation dataset generated by AssessPseudogenes unless it already is present.
stats_v#.RData = Pseudogene statistics generated by AssessPseudogenes unless it already is present.
breseq_[species] = Main breseq output.
